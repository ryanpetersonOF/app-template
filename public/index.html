Repro steps

  1. Click 'Cycle Listener' to add one 'application-started' and one 'application-closed' system event listener.
  2. Observe listener counts
  3. Click 'Launch App'
  4. Click 'Close App'
  5. Observe console logs. Observe timestamp updates.
  6. Rinse and repeat. Eventually, events will stop firing.

Notes: Adding these events manually in console will cause the issue to be unreproducible.
