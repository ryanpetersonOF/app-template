Problem

document.DOMContentLoaded events are not being fired on child window first page loads. Subsequent page loads (after reload) work without issue. Application primary windows do not seem to be affected by this issue.

Expected Behavior

document.DOMContentLoaded events are fired on first and subsequent page loads.

Repro

  1. Open console
  2. Observe this window background and console messages
  3. Click new child window button
  4. Observe child window console message
  5. Observe red background in child window
  6. Click reload button in child window
  7. Observe green background (changed in DOMContentLoaded event)
  8. Observe new child window success console message
