Problem

Creating multiple channels providers will cause all but the first providers clients onDisconnection listeners to fail.

Repro steps

  1. Launch Client
  2. Launch Provider (this creates multiple providers in code)
  3. Observe Green client backgrounds (connected to provider)
  4. Close Provider
  5. Observe single yellow background (onDisconnection fired)
  6. Observe onDisconnection listener never fired for clients (still green background)
  7. Remove channel.create() call in js/provider.js
  8. Rinse and repeat
  9. Observe client onDisconnection firing (red/yellow background)
